# 📈 Stock Price Prediction using LSTM & Sentiment Analysis

## 📌 Project Overview

This project predicts stock prices using **LSTM (Long Short-Term Memory)** and enhances predictions with **sentiment analysis from news data**.

---

## 🚀 Features

* 📊 Data Collection & Preprocessing
* 🧠 Sentiment Analysis using NLP
* 🔮 LSTM Model for Time Series Prediction
* 📉 Visualization of Predictions vs Actual Prices

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* TensorFlow / Keras
* Matplotlib, Seaborn
* NLP (TextBlob / Vader)

---

## 📂 Project Structure

* `data/` → raw & processed datasets
* `notebooks/` → Jupyter notebooks
* `src/` → source code (modular)
* `outputs/` → saved models & graphs
